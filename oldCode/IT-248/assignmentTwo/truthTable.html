A Truth Table Explained in English :)


A truth table is a sneaky little box that is fundamental to understanding how binary numbers, a.k.a 1's and 0's add up to each other. It may sound confusing but trust me, it'll make sense.

As you can see, there are four rows that have some kind of addition. In row 1, 0 + 0 = 0. In a ten number system or 0 to 9, the number system we grew up learning, 0 + 0 does equal 0.

The next row is 0 + 1. This is the 0 in the "2" column being inactive and the 1 activating the "1" column. The answer will be 1.

1 + 0 equals 1 or in normal terms, the answer is 2. Think back to the table we used on the Bit Table page. The 1 is showing that the column with the "2" is active and the 0 shows that the "1" column is inactive. Therefore, the answer is 2.

The last row is 1 + 1, the normal number system suggests that the answer is 2, however this is wrong. The first 1 is activating the "2" column and the second 1 is activating the "1" column. Therefore, the answer is 3.

After looking at the above table, the math can be applied to the other numbers on the bit table. With this concept, the number for example, 60 can be acquired by the number sequence 00111100.
